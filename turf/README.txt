This mod was inteded for the purpose of removing the fog, snow, and various other parts of Icebox to improve the all around performance of the map. 

Players should see an increase in stable FPS across the map.

To install this mod, just drag and drop the "turf" folder in your tagmods directory (Halo>mods>tagmods>turf).

After placing the mod, just run the install.bat

Done! Be sure to check out Icebox and make sure it installed correctly.
