EXTRACT INSTALLER TAGS
This batch file will extract all of the tags found in \h3mods\h3hud\tags from a provided build of Halo Online.
Overwrites provided tags, use with caution.

EXTRACT ALL MODIFIED TAGS
This batch file will extract all of the tags found in \h3mods\h3hud\tags, and modified tags in \h3mods\h3hud\scripts from a provided build of Halo Online.
Overwrites provided tags and scripts, use with caution.

GENERATE TAGTOOL SCRIPTS
This batch file will generate CMDS files based on differences between two provided builds of Halo Online.
Use a clean ElDewrito release and a release with the Halo 3 HUD installed for best results.
Overwrites provided scripts, use with caution.
